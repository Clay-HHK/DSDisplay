### 项目简介

本项目由C++和[EGE 20.08](https://xege.org/)构建，直观且美观地可视化常用数据结构——哈夫曼树、平衡二叉树和二叉排序树的构造和查询，并提供交互式菜单界面和简单的错误处理，可以用作数据结构课程设计。

**注意**：请使用**Release**模式编译，出于EGE库的未知原因，在启用多个绘图窗体时会出现段错误。
